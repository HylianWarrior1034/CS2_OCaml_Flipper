How to start frontend (Ensure you have [Rescript](https://rescript-lang.org/) installed.)
```
cd app
npm install --force
yarn start
```

How to start server
(Ensure you have [Dream](https://aantron.github.io/dream/) already)
```
cd server
dune build
dune exec ./app.exe (depends on your development environment)
```
